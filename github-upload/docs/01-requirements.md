# Requirements

## 서비스 요약

LIBCON은 사용자가 실제 도서관에 방문해 책을 읽고 인증하면, 개인 EXP와 소속 진영의 도서관 영향력을 얻는 위치 기반 독서 점령 서비스다.

## Core MVP

| 구분 | 기능 | 설명 |
| --- | --- | --- |
| 계정 | Google 로그인 | Google 계정으로 회원가입 및 로그인한다. |
| 계정 | 신규 사용자 판별 | 로그인 후 온보딩 완료 여부를 판단한다. |
| 온보딩 | 닉네임 설정 | 2자 이상 12자 이하, 한글/영문/숫자 허용, 중복 불가. |
| 온보딩 | 진영 선택 | 기존 진영 중 하나를 선택한다. |
| 온보딩 | 진영 생성 | 진영 이름, 대표 색상, 가입 방식, 소개를 입력해 생성한다. |
| 지도 | 내 위치 표시 | GPS 기반으로 현재 위치를 표시한다. |
| 지도 | 주변 도서관 표시 | 기본 반경 5km 이내 도서관을 지도 핀으로 표시한다. |
| 지도 | 점령 색상 표시 | 도서관 핀은 현재 점령 진영 대표 색상으로 표시한다. |
| 도서관 | 상세 조회 | 도서관 정보, 현재 점령 진영, 진영별 영향력, 거리 표시. |
| 독서 | 위치 체크인 | 도서관 반경 250m 이내에서만 독서 시작 가능. |
| 독서 | ISBN/바코드 스캔 | ISBN으로 도서 정보를 조회하고 저장한다. |
| 독서 | 타이머 | 최소 20분 이상 독서해야 인증 제출 가능. |
| 독서 | 위치 이탈 감지 | 독서 중 250m 범위 유지 여부를 주기적으로 확인한다. |
| 인증 | 감상평 검증 | 감상평이 도서 내용과 관련 있는지 AI와 로컬 규칙으로 검증한다. |
| 인증 | 페이지/위치 검증 | 읽은 페이지 수 이상치와 위치 로그를 검증한다. |
| 보상 | EXP 지급 | 최종 인증 성공 시 사용자 EXP 증가. |
| 보상 | 영향력 증가 | 소속 진영의 해당 도서관 영향력 증가. |
| 보상 | 점령권 갱신 | 영향력이 가장 높은 진영으로 도서관 점령권 갱신. |
| 마이페이지 | 내 서재 | 인증 성공한 도서와 감상평을 확인한다. |
| 랭킹 | 개인/진영 랭킹 | 개인 EXP, 진영 점령 도서관 수와 총 영향력 기준. |

## Nice-to-have

| 구분 | 기능 | 설명 |
| --- | --- | --- |
| 소셜 | 진영 채팅 | 같은 진영 사용자끼리 실시간 채팅. |
| 알림 | 영향력 역전 알림 | 점령권 위협 또는 역전 시 알림. |
| 마이페이지 | 독서 캘린더 | 월별 독서 기록 표시. |
| 마이페이지 | 칭호/업적 | 활동량에 따른 칭호와 배지. |
| 랭킹 | 지역별 랭킹 | 전국/지역 단위 랭킹. |
| 도서 | 소장 여부 확인 | 특정 도서관의 도서 소장 여부 조회. |
| 도서 | 추천 도서 | 인기 대출 도서 및 추천 도서. |

## 화면 단위 기능

| 화면 ID | 화면명 | 포함 기능 |
| --- | --- | --- |
| S1 | 스플래시 & 로그인 | 앱 로고, Google 로그인 버튼 |
| S2 | 온보딩 | 닉네임 입력, 중복 확인, 진영 선택/생성, 이용 안내 |
| S3 | 메인 홈 지도 | 내 위치, 주변 도서관 핀, 점령 색상 |
| S4 | 도서관 상세 | 도서관 정보, 영향력, 독서 시작 버튼 |
| S5 | 스캐너 뷰 | ISBN/바코드 스캔, 도서 정보 확인 |
| S6 | 독서 세션 | 타이머, 책 정보, 위치 상태, 인증 버튼 |
| S7 | 인증 및 검증 폼 | 시작/종료 페이지, 감상평 |
| S8 | 결과 화면 | 인증 결과, EXP, 영향력, 점령권 변경 |
| S9 | 랭킹보드 | 개인 랭킹, 진영 랭킹 |
| S10 | 마이페이지 | 프로필, 내 서재, 세션 기록, 기여 도서관 |
