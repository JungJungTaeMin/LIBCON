### 감상평 검증 기준

감상평은 Gemini 검증 결과와 로컬 규칙을 함께 사용한다.

로컬 규칙:

- 최소 30자 이상이어야 한다.
- 책 제목, 저자, 주제, 배운 내용, 개념, 문제, 시험/필기 관련 내용 등 책과 관련된 단서가 필요하다.
- 자기소개, 일상 잡담, 게임 이야기, 테스트 문구 등 무관 키워드가 포함되면 실패한다.
- Gemini가 통과해도 로컬 규칙을 통과하지 못하면 실패한다.

차단 예시:

```text
안녕하십니까 저는 정태민 입니다 하하
저는 집을 가고 싶네요. 오버워치 짱짱짱
```

## 현재 남은 제약

| 항목 | 설명 |
| --- | --- |
| 인증 방식 | 쿠키 세션 중심이며 JWT 중심 구조와 혼재 |
| 도서관 데이터 | 운영시간/휴관일은 외부 API가 제공하지 않으면 비어 있음 |
| Kakao Maps SDK | JS 키/등록 도메인 설정이 잘못되면 미니맵 fallback 표시 |
| 바코드 스캔 | 미지원 브라우저용 라이브러리 fallback 없음 |
| AI 검증 | Gemini 실패 시 로컬 규칙으로 fallback |
| 테스트 버튼 | `테스트 +10분`, 테스트 위치 지정은 개발 편의 기능이므로 배포 전 숨김 검토 필요 |
| 책 표지 인증 | 현재 사용자 요청에 따라 제거됨 |

## 최근 변경 요약

- 독서 시작 거리 제한을 100m에서 250m로 변경했다.
- 도서관 검색을 전체 거리 기준으로 바꾸고 검색 결과 리스트를 스크롤 가능하게 했다.
- 도서관 상세, 인증 페이지, 마이페이지를 스크롤 가능하게 했다.
- 독서 인증에서 책 표지 인증을 제거했다.
- 감상평 검증을 30자 이상 + 책 관련성 기준으로 강화했다.
- 마이페이지 독서 기록은 인증 성공한 세션만 표시한다.
- 내 서재 감상평은 3줄 축약 후 클릭으로 전체 보기/축약 토글한다.
